# 🐝 Make Bee Trip - Travel & Tourism Full Stack Project

A modern full-stack travel booking web application inspired by MakeMyTrip.

---

## 🚀 Features

- ✈️ Flight Booking (API Integration)
- 🚌 Bus Booking (API Integration)
- 🚆 Train Booking (API Integration)
- 🏨 Hotel Listing (API + Static Data)
- 🔍 Search-based results UI
- 📱 Responsive Modern UI (React JS)
- ⚡ Real-time backend API connection

---

## 🛠️ Tech Stack

### Frontend:
- React JS
- Axios
- CSS / Tailwind CSS (optional)

### Backend:
- Node.js
- Express.js
- CORS
