#  Make Bee Trip - Full Stack Travel Booking App

##  Overview
A full stack travel booking system built using React, Node.js, Express.

##  Features
- Flight Booking API
- Bus Booking API
- Train Booking API
- Hotel Listings
- Responsive UI

##  Tech Stack
Frontend: React.js  
Backend: Node.js, Express.js  

##  Setup

### Backend
cd backend  
npm install  
node server.js  

### Frontend
cd frontend  
npm install  
npm start
